# Legacy Scripts

This directory contains bare-metal deployment scripts used for manual multi-node SSH orchestration (exp01 through exp06, six physical machines with hardcoded IPs). These scripts have been superseded by the Docker-based workflow (`scripts/docker-deploy-federated.sh` and `.claude/commands/docker-queue.md`).

For current federated experiments, use the Docker automation. For reference to the legacy architecture, see `docs/QUICK_START_6NODES.md`, `docs/SETUP_6NODES.md`, and `docs/NODE_CONFIGURATION_MATRIX.md` (note: those docs were last updated for the legacy flow and may reference paths or configurations that have changed).
